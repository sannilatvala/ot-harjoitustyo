## Käyttöohje

Lataa projektin viimeismmän releasen lähdekoodi

### Ohjelman käynnistäminen

Asenna ensin riippuvuudet komennolla

```bash
poetry install
```
Käynnistä sitten peli komennolla:

```bash
poetry run invoke start
```
